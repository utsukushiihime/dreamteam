## Welcome to DreamTeam

#### Install npm files

```
npm i
```
#### Create a next.config.js file in the project root to connect to mysql database

```
module.exports = {
  env: {
    MYSQL_HOST: "",
    MYSQL_PORT: "3306",
    MYSQL_DATABASE: "",
    MYSQL_USER: "",
    MYSQL_PASSWORD: "",
  },
};
```

#### Run project

```
npm run dev
```
